## Getting Started

Edit this file or create new `.md` files in `src/content/blog/` — they automatically become blog posts.

Each post needs frontmatter with at least `title`, `description`, and `date`.

## Features

- SEO meta tags and Open Graph
- Structured data (JSON-LD)
- Auto-generated sitemap
- Ad placement slots
- Mobile responsive
- Deploys to Cloudflare Pages
